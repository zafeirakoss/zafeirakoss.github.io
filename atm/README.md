# ATM Simulation - C# Project

Αυτό το project είναι μια απλή προσομοίωση ATM γραμμένη σε C#.

## Τι κάνει
- Σύνδεση χρήστη με όνομα και κωδικό
- Έλεγχος υπολοίπου
- Ανάληψη χρημάτων
- Ανανέωση υπολοίπου μετά από κάθε συναλλαγή

## Τεχνολογίες
- C# (.NET Framework)

## Πώς να τρέξεις το project
1. Άνοιξε το αρχείο `ATM.sln` στο Visual Studio 2022 ή νεότερο.
2. Πάτησε Run (F5) για να ξεκινήσει η εφαρμογή.
